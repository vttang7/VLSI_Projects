# VLSI_Projects
This is my VLSI projects done through Cadence Virtuoso. In this project, I built everything from CMOS topology and using it as building blocks to create other complex digital combinatorial and sequential logic. 

Although VLSI typically focuses on building RTL and have layout automated, this project serves to focus on physical layout design and doing the layout by myself. To verify my work, testbenches was created inside Cadence Virtuoso and transient response was plotted to assess if the components behave as expected. 

This is my journey progression that I have completed:

- Project 1: Inverter and Ring Oscillator
- Project 2: CMOS Combinational Logic Gate Design
- Project 3: 4-bit Adder/Subtractor
- Project 4: 4-bit synchronous binary up/down counter
- Project 5: 32-bit ALU (My own personal project)
